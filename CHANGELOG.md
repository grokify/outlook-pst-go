## [v0.1.0] - 2026-05-16

### Highlights

- Pure Go library for reading and writing Microsoft Outlook PST files
- Full support for both ANSI and Unicode PST formats
- Transaction-safe write operations with two-phase commit

### Added

- Primitive types for PST parsing: NodeID, BlockID, HeapID, PropID, GUID ([`20dc5e4`](https://github.com/grokify/outlook-pst-go/commit/20dc5e4))
- Disk layer: header parsing, page/block structures, CRC computation, encryption/decryption ([`13d970b`](https://github.com/grokify/outlook-pst-go/commit/13d970b))
- NDB layer: node database with NBT/BBT B-tree search, node data reading, subnode support ([`74b938a`](https://github.com/grokify/outlook-pst-go/commit/74b938a))
- LTP layer: Heap-on-Node, BTree-on-Heap, Property Context, Table Context, named properties ([`fbe6445`](https://github.com/grokify/outlook-pst-go/commit/fbe6445))
- RTF decompression using LZFu algorithm for compressed message bodies ([`1188586`](https://github.com/grokify/outlook-pst-go/commit/1188586))
- Messaging layer: PST, Folder, Message, Attachment, Recipient types with lazy loading ([`76f57ca`](https://github.com/grokify/outlook-pst-go/commit/76f57ca))
- Search folder support for virtual folders with search criteria ([`76f57ca`](https://github.com/grokify/outlook-pst-go/commit/76f57ca))
- Disk layer write support: allocation maps, header/page/block serialization ([`e118f51`](https://github.com/grokify/outlook-pst-go/commit/e118f51))
- NDB layer write support: B-tree modifications, node creation/deletion ([`4dd6b53`](https://github.com/grokify/outlook-pst-go/commit/4dd6b53))
- Transaction system with two-phase commit and crash recovery ([`4dd6b53`](https://github.com/grokify/outlook-pst-go/commit/4dd6b53))
- LTP layer write support: heap allocation, BTH modifications, property bag updates ([`6a6a8f1`](https://github.com/grokify/outlook-pst-go/commit/6a6a8f1))
- Messaging layer write support: PST creation, folder operations, MessageBuilder API ([`65f7cf3`](https://github.com/grokify/outlook-pst-go/commit/65f7cf3))
- `pstinfo` CLI tool for inspecting PST file contents ([`3739ab6`](https://github.com/grokify/outlook-pst-go/commit/3739ab6))
- Go 1.23+ iterator support for idiomatic folder and message iteration ([`76f57ca`](https://github.com/grokify/outlook-pst-go/commit/76f57ca))
